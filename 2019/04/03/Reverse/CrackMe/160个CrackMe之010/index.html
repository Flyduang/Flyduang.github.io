160个CrackMe之010

发表于 2019-04-03 | 分类于 逆向分析 ， CrackMe

160个新手练习CrackMe，第10个。

0x1 环境

  • 系统：Windows XP SP3
  • 软件：Andrénalin.3
  • 工具：PEiD v0.95、吾爱破解专用版OllyDbg

0x2 分析

运行程序，注册码验证。

PEiD查壳，无壳，Microsoft Visual Basic 编写。

输入任意注册码，提示注册失败。

载入OD，确定失败字符串，定位关键跳转。

发现比较函数，加密后的真码为kXy^rO|*yXo*m\kMuOn*+。

向上分析，发现加密循环。

实际很简单，就是将输入注册码的每一位加上 0xA，再与正确的加密验证码比较。

通过反向计算，可以得到真正的注册码。

结果为：aNoThEr oNe cRaCkEd !。

进行验证。
